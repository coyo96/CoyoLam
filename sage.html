Sage is a fully integrated meal planning website that allows the user to:

  • Login or Register an account
  • Look through dozens of recipes
  • A fully functional search bar with optional selections
  • Favorite/ Unfavorite recipes
  • Create new recipes for the database
  • Add ingredients from the recipe to a grocery list
  • View, edit, and delete items in the grocery list
  • Add or edit user's recipes
  • Initate a pop up to read recipe directions step-by-step
